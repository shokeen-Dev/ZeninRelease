🌌 Zenin v1.0 — README

Zenin is a high-performance, minimalist anime streaming platform built for a premium viewing experience. Designed with a local-first philosophy, Zenin prioritizes user privacy and fluid performance by keeping your data on your device while providing a seamless connection to your favorite content. ⚡
🚀 Features

    ✨ Premium Minimalist UI: A sleek, Vercel-inspired interface designed for clarity and ease of navigation.

    🛡️ Local-First Experience: Your viewing history and preferences stay on your device—privacy is the priority.

    🎬 Dynamic Streaming: Powered by high-efficiency native modules for stable, high-quality playback.

    🧠 Smart Caching: Optimized data management that stores essential anime info while fetching fresh episode links on-demand.

    📱 Cross-Platform Architecture: Built using the Lynx framework for a truly native feel and performance.

📲 Installation

To install Zenin v1.0 on your Android device:

    📥 Download: Obtain the latest Zenin.apk from the official release section.

    🔓 Permissions: Enable "Install from Unknown Sources" in your device settings.

    📲 Install: Open the APK file and follow the on-screen prompts.

    ✅ Verify: The app includes built-in signature integrity checks to ensure you are running the official, secure version.
